# GoRedisManager
redis web客户端管理平台。优点：跨平台、部署简单

## 功能清单

:white_check_mark:管理连接、切换DB

:white_check_mark:支持 string 类型

:white_check_mark:支持 list 类型

:white_check_mark:支持 set 类型

:white_check_mark:支持 zset 类型

:white_check_mark:支持 hash 类型
